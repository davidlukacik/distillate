Distillate

Distill research papers from Zotero through reMarkable into structured notes.

Save paper in Zotero ──> PDF on reMarkable │ Read & highlight on tablet Move to Read folder when done │ Distillate picks it up: ├── Annotated PDF with highlights ├── Markdown note + AI summary ├── Reading log entry └── Paper archived on reMarkable
pip install distillate && distillate init

What it does

Highlight extraction Parses reMarkable .rm files to extract highlighted text, grouped by page
Annotated PDFs Overlays highlights on the original PDF using text-search matching
AI summaries One-liner, paragraph summary, and key learnings via Claude
Structured notes Markdown with metadata, highlights, and Obsidian or plain folder output
Reading log Auto-updated log of everything you've read, sorted by date
Email digest Weekly summaries and daily paper suggestions from your queue
